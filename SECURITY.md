# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue.** 

Instead, report it directly through the repository's dedicated issue tracker:
* **Report:** [https://github.com/n1nerlang/ditto/issues](https://github.com/n1nerlang/ditto/issues)

*Note: Please mark the issue as "Private" or "Confidential" if the reporting platform allows, or provide a high-level summary if necessary.*

## Project Status & Feeds

* **Ditto Feed:** Currently under development. Monitor [n1nerlang.github.io/ditto](https://n1nerlang.github.io/ditto) for future infrastructure updates.
* **Issue Tracking:** During this construction phase, all development updates and bug reports are being tracked via the [GitHub Issues](https://github.com/n1nerlang/ditto/issues) page. Please use this space for official project discourse until the Feed is fully operational.
